# Eight Counter - 仕様書

## 概要

8個の独立したカウンターを備えたPWA対応Webアプリケーション。各カウンターにユーザーが任意の名前を設定できる。

## 機能要件

### カウンター機能

- 画面上に8個のカウンターを表示する
- 各カウンターは独立して動作する
- 各カウンターの操作:
  - **+1**: カウントを1増やす
  - **-1**: カウントを1減らす（0未満も許容）
  - **リセット**: カウントを0に戻す
- 各カウンターに任意の名前（ラベル）を設定・編集できる
  - デフォルト名: 「カウンター1」〜「カウンター8」

### データ保存

- カウンターの値と名前はlocalStorageに自動保存する
- ブラウザを閉じても状態が維持される
- 全カウンター一括リセット機能

### PWA対応

- Service Workerによるオフライン動作
- マニフェストファイルによるホーム画面への追加対応
- アプリアイコンの設定

## 非機能要件

- レスポンシブデザイン（スマートフォン・タブレット・PC対応）
- 軽量・高速な動作（フレームワーク不使用、HTML/CSS/JS のみ）

## 技術構成

| 項目 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（バニラ） |
| データ保存 | localStorage |
| PWA | Service Worker + manifest.json |

## 画面構成

```
┌──────────────────────────────────┐
│         Eight Counter            │
├────────┬────────┬────────┬───────┤
│ [名前] │ [名前] │ [名前] │[名前] │
│   0    │   0    │   0    │  0    │
│ [+][-] │ [+][-] │ [+][-] │[+][-]│
├────────┼────────┼────────┼───────┤
│ [名前] │ [名前] │ [名前] │[名前] │
│   0    │   0    │   0    │  0    │
│ [+][-] │ [+][-] │ [+][-] │[+][-]│
└────────┴────────┴────────┴───────┘
         [全リセット]
```

- 4列 × 2行のグリッドレイアウト
- スマートフォンでは2列 × 4行に切り替え
